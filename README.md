# Multi-Author Blog Platform

## Tech Stack

- Next.js 15 (App Router)
- Prisma ORM
- PostgreSQL
- JWT Auth
- Tailwind CSS

## Features

- SSR & CSR hybrid rendering
- Role-based access control
- Author dashboard
- Admin moderation
- Full-text search
- Pagination
- Nested comments
